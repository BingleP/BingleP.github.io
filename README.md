# kerrick.ca — personal site

A retro Windows 98-style personal homepage for [kerrick.ca](https://kerrick.ca).

## Features

- Draggable, minimizable windows (profile, welcome, train photo, Steam)
- Animated floating clouds background
- Pastel colour theme with pixel-style fonts (VT323, Nunito)
- Taskbar with live clock
- No ads, no trackers, no hair

## Stack

Pure HTML, CSS, and vanilla JS — no frameworks, no build steps.
